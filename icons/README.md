# Iconos PWA

Para que las aplicaciones sean instalables, necesitas crear iconos en diferentes tamaños.

## Opción 1: Generador automático (recomendado)

Ejecuta el script de Python incluido:
```bash
python generate_icons.py
```

Este script creará iconos simples con el emoji/letra de cada app.

## Opción 2: Herramienta online

1. Ve a: https://www.pwabuilder.com/imageGenerator
2. Sube una imagen de 512x512 píxeles
3. Descarga el paquete de iconos
4. Coloca los archivos en las carpetas correspondientes:
   - `/icons/` - Para el sitio principal
   - `/calculadora-sublimacion/icons/` - Para la calculadora
   - `/piano/icons/` - Para la app de piano

## Tamaños requeridos

- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

## Opción 3: Usar un diseño personalizado

Si tienes un logo o diseño, puedes usar herramientas como:
- GIMP (gratis)
- Photoshop
- Figma
- Canva

Crea una imagen cuadrada de 512x512px y luego redimensiónala a los tamaños listados arriba.
